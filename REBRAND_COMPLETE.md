## Brand Identity Overview

### Organization Name
**Ujyalo Bhavishya Abhiyan**
- **Nepali:** उज्यालो भविष्य अभियान
- **English Translation:** "Bright Future Campaign"
- **Tagline:** "Safe Children. Bright Futures."

### Mission
Protecting Nepal's vulnerable children through institutional monitoring, community reporting, and direct action to ensure safety and opportunity.

### Color Palette

| Color | Hex | Meaning | Usage |
|-------|-----|---------|-------|
| **Deep Red** | #B31217 | Safety, Protection, Vitality | Primary buttons, headings, hero sections |
| **Gold** | #F5C542 | Prosperity, Enlightenment, Hope | Accents, borders, highlights |
| **Saffron** | #FF8C00 | Wellness, Courage, Sacrifice | Secondary accents, illustrations |
| **Dark Background** | #0B0B0B | Stability, Depth | Text, dark backgrounds |

**Key Feature:** All colors meet WCAG AAA contrast standards.

### Typography

**Nepali Text:**
- Font: Noto Sans Devanagari (Google Fonts)
- Example: सुरक्षित बालबालिका। उज्यालो भविष्य।

**English Headings:**
- Font: Montserrat (weights 400, 600, 700, 800)

**Body Text:**
- Font: Open Sans (weights 400, 600, 700)

**Fallback:** System font stack for universal compatibility

## Visual Identity Standards

### Hero Section
```
Background: Linear gradient (#B31217 → #FF8C00)
Text: White, bold headlines
CTA: Deep Red buttons with white text
```

### Navigation Bar
```
Logo: SVG icon + organization name
Colors: Deep Red primary, Gold accents
Font: Montserrat (headings), Open Sans (body)
Clear space: 20px minimum around logo
```

### Buttons
```
Primary: Background #B31217, text white
Hover: Darker red (#8B0D11), lift effect
Outline: Deep Red border with transparent background
Disabled: Gray background with gray text
```

### Color Combinations
```
✅ Deep Red + White: 7.2:1 contrast (WCAG AAA)
✅ Gold + Dark Background: 9.8:1 contrast (WCAG AAA)
✅ All text combinations: Minimum 4.5:1 (WCAG AA)
```
